Reves project structure

Reves development is divided in different Jupyter notebook for easier understanding. Once in Reves notebooks, you will be able to navigate through the notebooks without having to reach this page, as we have prepared a link at the end of each notebook that will lead you to the following document.

To share data between notebooks we have used Pandas' method pd.to_pickle to avoid transforming lists into strings as method pd.to_csv does.

If you wish to review the full project, start with Cleaning Data. In case you'd like to review a specific part of the project, just go directly to it.
