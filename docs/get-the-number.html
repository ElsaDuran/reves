Show me the money

How to use Reves?

Reves is a interactive web app where a production company can enter the main features of the movie they are about to produce and instantly know the potential revenue of the movie. As it is an interactive tool, you can play with: change the values, sumbit them and find the perfect featuring for your movie.

Which features can I play with?

Cast and crew

Name Description Posible Values
Director Name of the main director Text
Main lead genre What is the gender of the main character? 0=Female; 1=Male
Second lead genre What is the gender of the second main character? 0=Female; 1=Male

Movie plot

Name Description Posible Values
Belongs to collection Does the movie belong to a saga or franchise? 1=yes; 0=no
Movie genders Does the movie belong to each of the listed genders? 1=yes; 0=no

Production features

Name Description Posible Values
Original language Is it English or not? 1=yes; 0=no
Main production company What's the name of the main production company? Text
Number of production companies How many companies are producing the film? Number (int)
Runtime How much do you expect your movie to last? Ammount (minutes)
IMAX Are you launching an IMAX movie? 1=yes; 0=no
3D Are you launching an 3D movie? 1=yes; 0=no

Release date

Name Description Posible Values
Release Month When are you planning on releaseing the movie? 1-12
Release Weekday When are you planning on releaseing the movie? 1=Mon to 7=Sun
